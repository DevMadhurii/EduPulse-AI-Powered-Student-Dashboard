## 📌 Overview
**Pro Student Dashboard** is a complete, frontend‑only academic management system built for **hackathons** and **small institutions**. It provides separate portals for **Admin** and **Students**, with real‑time data persistence using browser `localStorage`.

## ✨ Key Features

### 👑 Admin Panel
- **Student Management** – Add, edit, delete, search (by name/roll), bulk CSV upload (50+ students).
- **Dynamic Subjects** – Add/edit/delete subjects; student marks adjust automatically.
- **Daily Attendance** – Date‑wise attendance marking with auto‑calculated %.
- **Assignments** – Create assignments, mark completion per student (matrix view).
- **Practicals** – Toggle completion for DAA, Python, VLab.
- **Timetable** – Edit day‑wise periods (time, subject, staff name).
- **Notices** – Add/Edit/Delete exam/event notices with release & last dates.
- **Analytics** – Class average bar chart, top performers, at‑risk students.

### 👨‍🎓 Student Dashboard
- **Profile** – Name, roll, email, contact, skills, fee status.
- **Performance Radar** – 5 criteria: Attendance, CT1, CT2, Mid Sem, Overall %.
- **Subject‑wise Marks** – CT1, CT2, Mid Sem for 7 default subjects (dynamic).
- **Timetable** – View day‑wise schedule with staff names.
- **Assignments** – See completion status (Complete/Incomplete).
- **Practicals** – Real‑time status of DAA, Python, VLab.
- **Notices** – Upcoming deadlines with date alerts.
- **AI Chatbot** – Ask natural language queries: *“my attendance?”*, *“weak subject?”*, *“exam tips?”*.
- **Smart Alerts** – In‑app + browser notifications for low attendance (<75%), fee pending, upcoming deadlines.

### 🎨 UI/UX
- **Dark Mode Toggle** – One‑click theme change.
- **Fully Responsive** – Works on desktop, tablet, and mobile.
- **Modern Glassmorphism Design** – Clean, professional look.
